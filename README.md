## wifi-password [arguments]

```diff

+>> --help: to get list f possible arguments

+>> --show-qr, -show: to show qr code 

+>> --save-qr [PATH], -save [PATH]: to save qr code in a folder

+>> --ssid [SSID], -s [SSID]: to get password of a particular SSID  
                     : for MAC users, need to specify SSID 
                     
+>> --list, -l: list all previous SSID's you were connected to  
              : shows with password in Linux and Windows  
              : shows without password in MAC(have to use "wifi-password -s [SSID]")
              
+>> --version: Show version number
